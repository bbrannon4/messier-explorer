# Messier Explorer

An interactive web app for exploring all 110 Messier deep-sky objects on an accurate sky chart with constellation context.

## Running Locally

Just open `index.html` in your browser — no server or install required. It's a fully static site.

## Features

- **Sky projections**: Switch between equirectangular, Mollweide, and stereographic views
- **Filtering**: Filter by object type, constellation, season, apparent magnitude, and angular size
- **Tonight's Sky**: Highlights objects currently above the horizon for your location
- **Click-to-detail**: Click any object for a side panel with description and Wikipedia data
- **Constellation context**: 15+ major constellations with lines, labels, and bright stars
- **Scale by magnitude**: Optionally size markers by apparent brightness

## Data

`Messier_data.csv` contains all 110 Messier objects with coordinates, magnitudes, distances, types, and best viewing seasons.
